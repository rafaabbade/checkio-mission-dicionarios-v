Existe um jeito mais fácil de criar os dicionários?

Sim! Você pode usar a função dict() para criar dicionários de forma rápida e eficiente, especialmente quando as chaves e valores já estão estruturados ou são conhecidos de antemão. Veja alguns exemplos de como dict() pode simplificar a criação de dicionários:

Argumentos Nomeados: Você pode passar pares chave-valor diretamente como argumentos nomeados, tornando o código mais legível: 

dicionario = dict(nome='Alice', idade=25, cidade='Nova York')
print(dicionario)
# Saída: {'nome': 'Alice', 'idade': 25, 'cidade': 'Nova York'}

Lista de Tuplas: Se você já tem dados em forma de lista de pares, pode convertê-los facilmente para um dicionário:

pares = [('nome', 'Alice'), ('idade', 25), ('cidade', 'Nova York')]
dicionario = dict(pares)
print(dicionario)
# Saída: {'nome': 'Alice', 'idade': 25, 'cidade': 'Nova York'}

Como calcular a média das notas de um dicionário?

Você pode usar a fórmula 'sum()' para somar todas as notas de um dicionário

Para acessar os valores do dicionário use dicionario.values().

Para garantir que a média das notas seja exibida com duas casas decimais, você pode usar formatação de strings ou a função round().
